# prakticna-provjera-json-2025

##  Opis zadatka

Napraviti web aplikaciju za pregledavanje informacija o predmetima u školi uz mogućnost dodavanja novih i filtriranja postojećih, sve to koristeći PHP, Bootstrap i JSON datoteku za pohranu podataka.

Kao pomoć u rješavanju zadatka ponuđena su već dva riješena zadatka sa cijelim izvornim kodom koji vam trebao biti od pomoći prilikom rješavanja ove praktične provjere.
